Data

I built a dataset of mayors for all Russian city districts (gorodskie okruga) for the period of 2011-2018. This dataset tracks the timeline of the city charters' changes and includes information about changes in the mayors' selection rule for every city, provides basic information of mayors' candidacies and tracks the switches of those candidacies for every city.

My primary programming languages are Python and R.

I wrote Pyhon scraping codes to extract municipal-level budget data and some R codes to build maps of Russian cities. These codes are available on my github page.

I also wrote Python codes to parse Russian procurement data. Those codes are availbale upon request.
